Ufopa.ThemeMandube Installation
-------------------------------

To install Ufopa.ThemeMandube using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``Ufopa.ThemeMandube`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        Ufopa.ThemeMandube
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
